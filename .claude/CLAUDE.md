# Claude Code Interactive Course

This is an interactive course for learning Claude Code. The course is designed to be taken inside a Claude Code terminal session.

## Course Structure

- **Lesson content**: `lesson-modules/*.md`
- **Slash commands**: `.claude/skills/lesson-*/SKILL.md`

## Available Commands

| Command | Description |
|---------|-------------|
| `/course` | Show course overview and progress |
| `/lesson-01` | Getting Started |
| `/lesson-02` | Context Management |
| `/lesson-03` | CLAUDE.md - Persistent Memory |
| `/lesson-04` | Skills - Custom Slash Commands |
| `/lesson-05` | Subagents - Delegating Tasks |
| `/lesson-06` | Hooks - Workflow Automation |
| `/lesson-07` | Advanced Workflows |
| `/lesson-08` | Best Practices |

## How This Course Works

1. Students type `/course` to see the course overview
2. Type `/lesson-XX` to start a lesson
3. Claude reads the lesson content and teaches it interactively
4. Each lesson has practice exercises
5. Students can ask questions about the material

## Teaching Instructions

When teaching a lesson:

1. Read the lesson file from `lesson-modules/`
2. Present the content in digestible sections
3. Pause for questions after major concepts
4. Encourage hands-on practice with the exercises
5. Summarize key takeaways at the end

## Important Notes

- This is a learning environment - be patient and encouraging
- Explain concepts clearly with examples
- Let students practice the commands in real-time
- Answer questions about the material
- Guide students to the next lesson when ready
