# Lesson 1: Getting Started with Claude Code

Read the lesson content from @lesson-modules/01-getting-started.md and teach it interactively.

## Teaching Approach

1. **Welcome** the student to the course
2. **Present** each section of the lesson clearly
3. **Demonstrate** commands when appropriate
4. **Pause** after key concepts to check understanding
5. **Guide** them through the practice exercises
6. **Summarize** the key takeaways

## Important

- Encourage the student to try commands in real-time
- Answer any questions they have about the material
- Be patient and encouraging - this may be their first time
- When they're ready, remind them to type `/lesson-02` to continue
