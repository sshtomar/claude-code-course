# Claude Code Interactive Course

Welcome! This course teaches you Claude Code through hands-on lessons.

## Course Outline

| # | Lesson | Command | Topics |
|---|--------|---------|--------|
| 1 | Getting Started | `/lesson-01` | Installation, basic usage, first code change |
| 2 | Context Management | `/lesson-02` | Context window, /clear, /compact, subagents |
| 3 | CLAUDE.md | `/lesson-03` | Persistent memory, project configuration |
| 4 | Skills | `/lesson-04` | Custom slash commands, reusable workflows |
| 5 | Subagents | `/lesson-05` | Task delegation, parallel work, isolation |
| 6 | Hooks | `/lesson-06` | Automation, pre/post tool hooks |
| 7 | Advanced Workflows | `/lesson-07` | Plan mode, worktrees, headless mode |
| 8 | Best Practices | `/lesson-08` | Patterns for great results |

## How to Use This Course

1. **Start a lesson**: Type `/lesson-01` to begin (or any lesson number)
2. **Follow along**: Try the commands as you learn them
3. **Do the exercises**: Each lesson has hands-on practice
4. **Ask questions**: Feel free to ask about anything unclear
5. **Move on**: Type the next `/lesson-XX` when ready

## Quick Start

Type `/lesson-01` to begin with the basics!
